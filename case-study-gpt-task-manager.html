1 The Problem

Every project starts as a vague idea. Turning that idea into an actionable task list takes a planning session, a meeting, or a lot of solo thinking — and then someone has to manually enter everything into a spreadsheet or project tool. This translation from idea to structured tasks is slow and often incomplete.

The goal was to build an agent that could take a rough project description through conversation, ask the right clarifying questions, and then automatically generate and populate a structured task list — no manual data entry required.

2 How It Works

The agent engages in a short back-and-forth conversation to gather project details, then executes automatically once it has what it needs.

Workflow Flow

Chat Trigger (User describes project) → Project Manager Agent (GPT-4.1-mini) → Have All Info? → Get More Info (if needed) → Structured Output Parser → Write JSON → Add Tasks to Google Sheets → Respond Complete

The Project Manager Agent uses Chat Memory to track the conversation. A "Have All Info?" logic node checks whether the agent has gathered enough detail — if not, it loops back to ask a follow-up question. Once satisfied, it runs the project description through a Structured Output Parser to extract task names, descriptions, priorities, and owners as clean JSON. Each task is then added as a row in Google Sheets automatically.
