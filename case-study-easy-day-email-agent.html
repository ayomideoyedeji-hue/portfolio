1 The Problem

Email is one of the biggest drains on a professional's day. Most emails follow predictable patterns — scheduling requests, follow-ups, confirmations, FAQs — yet each one demands manual attention. The average knowledge worker spends over 2 hours a day in their inbox, much of it on responses that are formulaic.

The goal was to build an agent that could handle routine email autonomously — reading context, drafting appropriate responses, managing calendar implications, and logging everything — without the user needing to open their inbox for every message.

2 How It Works

The workflow triggers instantly on every new incoming email and hands it off to an AI agent with access to Gmail, Google Calendar, and memory tools.

Workflow Flow

New Email Received → Email Assistant Agent (GPT-4o) → Gmail Tool Google Calendar Tool Simple Memory → Log to Google Sheets → Slack Notification

The AI agent receives the full email and decides what action to take. It has tools available to it: Gmail Tool to draft and send replies, Google Calendar Tool to check availability and create events, and Simple Memory to maintain context across a conversation thread. Every action is logged to Google Sheets and a Slack notification is sent so the user stays informed without being in the loop for every email.
